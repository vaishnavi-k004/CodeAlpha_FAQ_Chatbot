# AlphaIntel | AI-Powered Internship Assistant

AlphaIntel is a generalized FAQ chatbot designed to assist interns with the CodeAlpha internship process. It uses Natural Language Processing (NLP) to understand user intent and provide accurate information regarding timelines, task submissions, and certification.

- **Intuitive UI**: A user-friendly interface built with Tkinter featuring clear navigation and input guidance.
